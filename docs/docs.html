Timer

Timer()

Constructor.

start()

Start the timer.

stop()

Stop the timer.

reset()

Reset the start and stop time.

time()

Returns the number of milliseconds that has elapsed between start and stop was called respectively. If the timer is running, the currently elapsed time will be returned. If the timer has not been started '0' will be the value.

Returns
Elapsed milliseconds as int.

Date Formats

Based on https://docs.python.org/3.5/library/datetime.html#strftime-strptime-behavior

Date

Format Description Example
%a Weekday, abbreviated.
%A Weeday, full name.
%w Weekday as a decimal number where 0 is Sunday and 6 is Saturday.
%d Day of the month as a zero-padded decimal number.
%b Month, abbreviated.
%B Month, full name.
%m Month as a zero-padded decimal number.
%y Year without century as a zero-padded decimal number.
%Y Year with century as a decimal number.

Time

Format Description Example
%H Hour (24-hour clock) as a zero-padded decimal number.
%I Hour (12-hour clock) as a zero-padded decimal number.
%p AM and PM
%M Minute as a zero-padded decimal number.
%S Second as a zero-padded decimal number.
%f Millisecond as a decimal number.
